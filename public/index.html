6SR Stellar to Fantom Gateway

This tool allows users to bridge the WFTM token between the stellar network (stellar.org) and the Fantom Opera network (fantom.foundation).

How to use it:
  1. Register your address using the form below.
  2. Check the current pool exchange rate to confirm that you have acceptable market conditions.
  3. Send the asset. This can be done with:
    • Send WFTM to the specified XLM address, and be SURE to include the memo.
    • Use the "Swap" contract command for ssrwFTM. Make sure to include the memo in the "memo (string)" field. Also remember that the amount is multiplied times 1018, so for 1 WFTM swap the "amount" will be 1000000000000000000.
